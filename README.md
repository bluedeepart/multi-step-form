## Overview

### The challenge

Users should be able to:

- Complete each step of the form seamlessly
- Navigate between steps without losing entered data
- See a summary of their selections before final submission
- Confirm their order on the final step
- Experience responsive design across devices
- View hover and focus states for interactive elements
